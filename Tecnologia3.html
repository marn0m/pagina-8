FUNCIONES DE EXCEL

Las funciones en Excel son fórmulas predefinidas que permiten realizar cálculos complejos de manera rápida y eficiente. Cada función tiene un propósito específico y se utiliza para manipular datos de forma automática. Las funciones simplifican tareas como la suma, el promedio, la búsqueda de datos, y mucho más, sin necesidad de escribir fórmulas largas manualmente.

EJEMPLOS:
1. Función SUMA
Descripción: La función SUMA agrega todos los números dentro de un rango de celdas. Es útil para calcular totales en listas de datos.



2. Función PROMEDIO
Descripción: La función PROMEDIO calcula el promedio aritmético de un rango de celdas, proporcionando una medida central de los datos.







3. Función MAX
Descripción: La función MAX devuelve el valor más alto de un rango de celdas, útil para identificar los valores máximos en un conjunto de datos.







4. Función MIN
Descripción: La función MIN devuelve el valor más bajo de un rango de celdas, útil para identificar los valores mínimos en un conjunto de datos.







5. Función BUSCARV
Descripción: La función BUSCARV busca un valor específico en la primera columna de un rango y devuelve un valor en la misma fila desde una columna especificada.
